WEB 2.0

Llegado el nuevo milenio, internet no solo se habia vuelto bastante mas accesible sino tambien mas conocido para el ciudadano promedio. Pero el 2001 sacudio al mundo. Las crisis financieras, y el terrorismo cambiaron para siempre la forma de ver la realidad y requirieron grandes cambios y avances para recuperarse, tanto a nivel economico como social y tecnologico. La crisis de empresas derivada de los problemas financieros llevaron a la evolucion de la web. Entonces, se busco que el usuario interactuara mas con ella. Esta nueva web 2.0(nacida oficialmente en 2004) busco que el usuario no solo consumiera, sino que tambien produciera(comentando, creando contenido, catalogando, respondiendo, etc.)

En este punto, el mayor productor de contenido paso a ser el usuario comun y corriente.
